## About
コーディネートがファッションコーディネートサイト[WEAR](https://wear.jp/)に投稿した際に何いいね付くかを予測してくれるアプリです。また採点結果はSNSでシェアすることができます。

## 使用技術

* Keras（モデル作成）

* Swift（アプリ開発）
